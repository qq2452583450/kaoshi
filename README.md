# 物资岗位考试系统

这是一个用于项目物资管理岗位培训的本地网页考试系统，支持电脑端和手机端自适应访问。

## 功能

- 员工自助注册，等待管理员审核。
- 管理员审核人员。
- 员工练习、正式考试、查看个人成绩。
- 客观题自动判分。
- 简答题和案例题生成建议分，管理员复核最终分。
- 管理员查询成绩并导出 CSV。
- SQLite 本地保存数据。

## 依赖

本项目使用 Python、Flask、python-docx、pytest 和 SQLite。

安装依赖：

```bash
pip install -r requirements.txt
```

## 启动

```bash
python app.py
```

浏览器打开：

```
http://127.0.0.1:5000
```

## 默认管理员

- 账号：`admin`
- 密码：`admin123`

## 数据位置

```
data/exam_system.sqlite3
```

系统会从项目根目录的 `.docx` 文件导入试卷。当前 Word 文件第五套在中途截断，缺少第五套后续题目和参考答案；因此本地演示会先导入前 4 套完整试卷，并在页面显示提示。替换为完整 Word 文件后，可导入完整 5 套。

## 安全特性

- **密码安全**：使用 PBKDF2 算法加盐哈希，迭代 100,000 次
- **CSRF 保护**：所有 POST 请求自动验证 CSRF token
- **SECRET_KEY**：从环境变量读取或自动生成随机密钥

## 环境变量

| 变量 | 说明 | 默认值 |
|------|------|--------|
| `SECRET_KEY` | Flask 会话密钥 | 自动生成随机值 |

## 第一版限制

- 暂无短信验证。
- 暂无企业单点登录。
- 暂无高级防作弊监考。
- 完整 5 套导入依赖完整 Word 文件。
